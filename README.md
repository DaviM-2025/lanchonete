# lanchonete
Estruturando o Projeto

Objetivo: entender a interação de classes e objetos em uma arquitetura MVC por meio do desenvolvimento de uma aplicação básica que simule uma lanchonete

Requisitos:
- Estudar o que é MVC e saber explicar o propósito de cada sessão dentro do modelo
- Criar entidades pedido, cliente, lanche e conta
- Criar classes cozinha e balcão de atendimento
- Cada lanche deve ter uma descrição do item e um preço
- Cada pedido deve ter ao menos um item e um cliente
- Cada cliente deve ter um nome
- Após fazer o pedido, o lanche deve entrar em uma fila de processamento para ser enviado à cozinha
- Após ser processado na cozinha, o pedido deve ser adicionado em uma lista para ser enviado ao balcão de atendimento
- A conta deve conter o valor total do pedido e o status atual
- A seleção de lanches para um pedido deve ser feita em um terminal de comando
- O código deve separar a entrada de dados, a lógica da aplicação e as entidades do sistema em diferentes arquivos de maneira clara e coerente
- O projeto deve ser enviado para o github usando comandos git
